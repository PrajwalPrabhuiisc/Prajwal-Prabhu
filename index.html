Relevant Projects

Contextual Reasoning Divergence Framework (CRDF)

Lightweight visual reasoning with Situation Awareness on CIFAR-10/Fashion-MNIST using ResNet-18, custom ViT, fine-tuned CLIP. Added predictive uncertainty, attention-guided perturbations, Grad-CAM, and Monte Carlo dropout for explainable failure analysis.

Ultra-Lightweight CLIP-FBS Recommendation System

Symbolic-multimodal system fusing CLIP vision-language embeddings with FBS knowledge graph and adaptive GNN. Real-time user feedback updates graph edges; includes counterfactual explanations.

Attention-Guided Active Learning for Vision-Language Models

Hybrid sampling strategy combining prediction uncertainty and attention-driven diversity. Attention-regularised fine-tuning improves cross-task adaptability.
