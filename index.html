Relevant Projects

Contextual Reasoning Divergence Framework (CRDF): A Lightweight Visual Reasoning approach

  • Implemented CRDF with Situation Awareness on CIFAR-10/Fashion-MNIST using CNN, ResNet-18, custom ViT, and fine-tuned CLIP; added predictive uncertainty, attention-guided perturbations, and feature stability for robust edge-case detection.
  • Enhanced reasoning divergence (RDS) via iterative local/global masking with attention prioritization; Integrated Grad-CAM, ViT/CLIP attention maps, and Monte Carlo dropout for explainable failure analysis
  • Optimized for Colab GPU; mixed precision, batch size 16, 3 epochs, memory cleanup; Utilized diagnostic framework.

Ultra-Lightweight CLIP-FBS Recommendation System for CPU-based Environments

  • Developed a symbolic-multimodal recommendation system integrating real CLIP (ViT-B/32) for vision-language embeddings with a Function-Behaviour-Structure (FBS) knowledge graph and adaptive GNN to fuse symbolic reasoning with neural similarity for interpretable product recommendations.
  • Implemented hybrid neural-symbolic retrieval combining CLIP-based cosine similarity with symbolic logic constraints (e.g., "wearable & durable") and counterfactual explanations, enabling filtered, explainable recommendations across 25 real-image products.
  • Engineered feedback-driven graph adaptation using user likes/dislikes to dynamically add edges in the FBS graph, visualized via NetworkX, with on-the-fly GNN retraining and re-encoding to reflect evolving user preferences in real time.

Attention-Guided Active Learning for Vision-Language Models

  • Proposed an attention-guided active learning framework applicable to transformer-based vision-language models.
  • Designed a hybrid sampling strategy that integrates prediction uncertainty with attention-driven diversity for efficient data selection.
  • Implemented attention-regularised fine-tuning to improve model adaptability across tasks and domains.
