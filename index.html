About Me

“I would rather have questions that can’t be answered rather than answers that can’t be questioned” – Richard Feynman

Relevant Projects

Contextual Reasoning Divergence Framework (CRDF)

  • Lightweight visual reasoning with Situation Awareness on CIFAR-10/Fashion-MNIST using ResNet-18, custom ViT, fine-tuned CLIP
  • Added predictive uncertainty, attention-guided perturbations, Grad-CAM, Monte Carlo dropout for explainable failure analysis
  • Optimized for Colab GPU with mixed precision and diagnostic framework

Ultra-Lightweight CLIP-FBS Recommendation System

  • Hybrid neural-symbolic system fusing CLIP (ViT-B/32) with FBS knowledge graph and adaptive GNN
  • Real-time user feedback updates graph edges; supports counterfactual explanations
  • Runs efficiently on CPU; tested on 25 real-image products

Attention-Guided Active Learning for Vision-Language Models

  • Hybrid sampling using prediction uncertainty + attention-driven diversity
  • Attention-regularized fine-tuning for cross-task adaptability
